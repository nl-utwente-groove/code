The lib folder includes the following Java archives:
  • ltl2ba4j.jar: Java archive for the algorithm to transform LTL formulea to Buchi automata, developed by Gastin.
  • ltl2buchi.jar: Java archive for the algorithm to transform LTL formulea to Buchi automata, developed as part of the Java PathFinder project.

LTL formulea transformers

Currently, GROOVE uses two external libraries to transform LTL formulea to Buchi automata. One is based on the algorithm is developed by Gastin and publiced at CAV'01. This website provides an online demonstration environment. A Java bridge for this tool is available from here. The other algorithm is developed as part of the Java PathFinder project. The Java sources of this algoritm are available from the following Subversion repository: https://javapathfinder.svn.sourceforge.net/svnroot/javapathfinder/, under the substructure trunk/extensions/LTL2Buchi.
