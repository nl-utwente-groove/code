This package contains basic functionality to deal with graphs. To start with, there is a hierarchy of interfaces and default implementations for graph elements, beginning with the top-level Element with sub-interfaces Node and Edge. Built on top of this, the interfaces GraphShape and Graph provide essential graph functionality. Of Graph there are a number of different implementations, based on different storage methods. Finally, there are provisions for mappings between graphs, in the form of interfaces NodeEdgeMap and Morphism, again with default implementations.
